# Flask Application

## Setup

### Prerequisites

- Python 3 installed on your system.

### Steps

1. **Clone the Repository**

   ```sh
   git clone <your-repo-url>
   cd <your-repo-name>

2. **Create Virtual Environment**

    ```sh
    python3 -m venv .venv

3. Activate the Virtual Environment

    On macOS and Linux:

    ```sh
    source .venv/bin/activate

4. Install Flask

    ```sh
    pip3 install flask
